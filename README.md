# 📝 Notes App 

## 📌 Overview
A simple and efficient Notes Application that allows users to create, edit, and delete notes with persistent storage.

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Local Storage

---

## ✨ Features
- 📝 Create, edit, delete notes
- 💾 Persistent storage (Local Storage)
- 🔍 Search notes
- 📱 Responsive UI
- 🌙 Dark mode

---

## 🧠 Key Learnings
- Local storage handling
- CRUD operations
- UI state management

---

git clone https://github.com/yuvraj0651/Notes-App.git
